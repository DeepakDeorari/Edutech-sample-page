Top 22 HTML Interview Questions and Answers

Here you are going to learn some most frequently asked html questions in the interviews.

Below are some top selected questions on html to crack technical interviews. These questons are selected based on beginner level to intermediate level and intermediate level to advance level.

HTML Questions:

Q1. What is HTML?


HTML stands for Hyper Text Markup Language. HTML is a text formatting language which is used to create and display pages on the Web. HTML makes the text more interactive and dynamic. Hypertext is the text which includes texts, images, links etc. and helps to display them on the browsers window. You can save your html files to web with the help of .html or .htm extension.

Q2. What is the difference between HTML elements and tags?

Elements Tags
Elements itself has a specific meaning in a section of a web page like an image or a link are elements. Tags are used to place the content. Tags are always define using ( <>)


Q3. What are Attributes in HTML?


Each tag has somee additional attributes which help us to alter the behaviour of the tag so that it looks more attaractive. Attributes can only be set inside opening tags.
Example...
You can set attributes to an <input> tag as text field, radio button etc.

Q4. What are the types of lists used in html?
OR
Name some common lists that are used when designing a page.


The lists that are used while design a page are:
  • Ordered list - Elements displayed in a numbered format. Repersented by <ol>.
  • Unordered list - Elements displayed in a bold black dot format. Repersented by <ul>.
  • Defination list - Elements displayed like a dictionary form. Tags used are <dl>, <dt>, <dd>.

Q5. How to create a nested webpage in HTML?
OR
What is the use of iframe tag.


The HTML iframe tag is used to display a nested webpage. iframe tag help us to include a web page inside another web page.
Example... <iframe src="<a href="https://www.google.com/">google</a> height="300" width="400" ></iframe>.

Q6. What is an Anchor tag in HTML?


Anchor tag <a> is a tag used to link two web pages with the help of "href" attribute.
Example...
<a href="https://www.google.com/"> click here </a>.

Q7. What is a marquee?


Marquee is used for the scrolling text on a web page. It scrolls the image or text up, down, left or right automatically. You have to put the text which you want to scroll within the <marquee> Scrolling text </marquee> tag.

Q9. How to make a picture of a background image of a web page?


To make a picture a background image on a web page, use following code:
<body background = "image.gif">
You can replace the "image.gif" with the name of image file which you want to display on your web page.

Q10. What happens if you open the external CSS file in a browser?
OR
Can you open an external CSS file in a browser?


It is not possible to open a file having .css extension in browser. You can only use a file with .css extension by reference it within a HTML document.

Q11. How do you create text on a webpage that allows you to send an email when clicked?


You need to use mailto command within the href tag to change the text into the clickable link to send an email.
Example...
<a href="mailto:your-email-address">text to be clicked</a>

Q12. How are active links different from normal links?

Both of the link have blue color but some browsers recognize an active link when the mouse cursor is placed over that link. Whereas, others recognize active links when the link has the focus.
Those links that do not have a mouse cursor over that link is considered a normal link.

Q15. How many types of CSS can be included in HTML?

We can include CSS in html by three ways:

Inline CSS: It is used for styling small contexts. To use inline styles add the style attribute in the relevant tag.

<p style="font-size:20px;"> this is a paragraph </p>

External Style Sheet: This is used when the style is applied to many pages. Each page must link to the style sheet using the tag. The tag goes inside the head section.

<head>
<link rel="stylesheet" type="text/css" href="mystyle.css" />
</head>

Internal Style Sheet: It is used when a single document has a unique style. Internal styles sheet needs to put in the head section of an HTML page, by using the <style> tag in the following way:

<head>
<style type="text/css">
hr {color:sienna}
p {margin-left:20px}
body {background-image:url("images/back40.gif")}
</style>
</head>

Q18. What are void elements in HTML?

HTML elements which do not have closing tags or those that don't need to be closed are Void elements.
For Example <br />, <img />, <hr />, etc.

Q19. What is the difference between link tag <link> and anchor tag <a>?

The anchor tag <a> is used to create a hyperlink to another webpage or to a certain part of the webpage. These links are clickable.
link tag <link> defines a link between a document and an external resource and these are not clickable.

Q20. Why Meta tags are used in HTML?

Meta tags in HTML are used by the developer to tell the browser about the page description, author of the template, character set, keywords and many more.
